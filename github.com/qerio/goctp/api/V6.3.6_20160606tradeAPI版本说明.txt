变更说明：
1.修复api如果多次断线可能导致占用内存过多而崩溃的问题。
2.交易类接口中新增字段LongCustomerName,traderapi涉及到的函数见下文（mduserapi没有涉及到） 

///长客户姓名
TThostFtdcLongIndividualNameType	LongCustomerName;
typedef char TThostFtdcLongIndividualNameType[161];
